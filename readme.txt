Pauthors contributors list Plugin.

== Description ==

Pauthors lets you select the authors from post edit page. You can select by checking the checkbox corresponding to each user under post.
Selected authors list will display in front end under post with a link to author page.

Major features in Nalert include:

* Can select single and mutiple user.
* Will display an extra block in front end under selected post.

== Installation ==

Upload the Pauthors plugin to your blog, Activate it, then got to post edit page and submit rquired Information.

You're done!
